### Commands
| Command              | Action                 |
|----------------------|:-----------------------|
| `/homes` `/home`     | Open your Home Gui     |
| `/home <1-7>`        |
| `/home set <1-7>`    | Set a home directly    |
| `/home delete <1-7>` | Delete a home directly |
| `/home remove <1-7>` | Delete a home directly |


### Admin Commands
| Command                       |
|-------------------------------|
| `/home <player> <1-7>`        |
| `/home <player> set <1-7>`    |
| `/home <player> delete <1-7>` |
| `/home <player> remove <1-7>` |
